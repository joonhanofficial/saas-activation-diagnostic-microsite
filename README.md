# SaaS Activation Diagnostic Microsite

## Purpose
A one-page interactive microsite built for the analytics case study: **"When SaaS Activation Looks Stable Until You Segment It"**. 
This project serves as an interactive proof asset to make the case study easier to inspect, understand, and share.

## Files Created
- `index.html`: The main structure of the microsite.
- `style.css`: The styling rules, establishing a dark, analytical, and premium aesthetic.
- `script.js`: Interactivity including smooth scrolling, counter animation, and mobile tap reveal logic.
- `README.md`: This documentation file.

This is a static GitHub Pages compatible microsite built using plain HTML, CSS, and JavaScript with no external dependencies or build tools.
To preview locally:
1. Clone the repository.
2. Open `index.html` in your web browser.

## Note
To make the site live, GitHub Pages needs to be manually enabled in the repository settings, pointing to the `main` branch.
